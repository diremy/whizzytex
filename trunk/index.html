WhizzyTeX

An Emacs minor mode for
incremental viewing of
LaTeX documents.

WhizzyTeX provides a minor mode for Emacs or XEmacs, a (bash) shell-script daemon and some LaTeX macros. It works under Unix with gv and xdvi viewers, but the Active-DVI viewer will provided much better visual effects and will offer more functionalities.

Versions

The current version is whizzytex-1.00.tgz

Documentation

Installation and manual are available in HTML and DVI formats.
Most of the documentation is also available as online Emacs help.
See also the FAQ.

Disclaimer

WhizzyTeX is still under development and may contain dangerous bugs. You may wish to save copies of the file (and of the whole directory you are working in) before WhizzyTeX-ing.

Viewers

One of the problems that you will encounter with WhizzyTeX, especially if you have a relatively old slow machine is that the previewer flashes whenever refreshing the page. xdvi is much faster but does not render colors and messes up with most inlined-Postscript specials. Conversely, gv is much slower, but more compliant. Unfortunately, both flash when the page is refreshed. Thus, we recommend the use of advi (Active-DVI) -see below.

Alternatively, there is a patched version of gv (version 3.5.8) written by Fabrice Le Fessant that inhibits flashing by drawing pages in a pixmap and flushing the page all at once. Here are the patched version (and its original). To remove flashing, gv must then be called with the option -nopixmap.

Active-DVI (hightly recommended)

This is a previewer initially based on Mldvi, written Alexandre Miquel and extended with many other features (correct handling of colors, of most inlined-Postscript specials, hyperrefs, pauses, replays, etc.), and in particular, with the ability to draw in the background and flush a page all at once, so as to avoid any flashing effect. Moreover, advi offers can point back to source files, as long as the dvi has been (automatically) instrumented with source file and line information (as specials). See the Active-DVI distribution.

Good options for use with WhizzyTeX are: 

    advi -html Start-Document OTHER-OPTIONS FILENAME
